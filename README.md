# NestJS Boilerplate

TypeORM, Mongoose, Redis, RabbitMQ, AWS-SDK, Slack-api and some utils like pipes,interceptors or guards can be used.

## To Start

### Environment

env.production, env.development, env.private, env.public can be used.

### Dependencies

`Node` 16  
`NestJS` 9  
`MySQL`  
`Mongoose`  
`RabbitMQ`  
`Redis`

### Scripts

Installation

```
$ npm ci
```

Build

```
$ npm run build
```

Start

```
npm run dev  #in dev
npm start  #in prod
```
